# Aster Lang 社区

> 帮 Policy-as-Code 支持您的母语。

## Lexicon 生态是什么？

Aster Lang 的 CNL 编译器**与语种无关**。每个支持的语言都是一个 **lexicon 插件**——含 keyword 翻译 + 语法规则的 JSON + Java 小包。

这意味着您可以：

- 用 ~75 个 keyword 翻译，把 Aster 接到您的母语
- 把您的 lexicon 发布到 maven artifact
- 成为 Aster Language Steward
- 获得每年 ¥3,000 platform credit（Steward 限定）

## Was ist das Lexikon-Ökosystem?

Der CNL-Compiler von Aster Lang ist **sprach­agnostisch konzipiert**. Jede unterstützte Sprache wird als **Lexikon-Plugin** ausgeliefert — ein kleines JSON + Java-Paket mit Schlüsselwort-Übersetzungen und Syntaxregeln.

Das bedeutet, Sie können:

- Aster mit ~75 Schlüsselwörtern in Ihre Sprache übersetzen
- Ihr Lexikon als Maven-Artefakt veröffentlichen
- Aster Language Steward werden
- Jährlich ¥3.000 Plattform-Guthaben verdienen (Steward-exklusiv)

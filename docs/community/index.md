# Aster Lang Community

> Help bring Policy-as-Code to your native language.

## What is the lexicon ecosystem?

Aster Lang's CNL compiler is **language-agnostic by design**. Every supported language ships as a **lexicon plugin** — a small JSON + Java package defining keyword translations and syntax rules.

This means you can:

- Translate Aster to your language with ~75 keywords
- Publish your lexicon as a maven artifact
- Become an Aster Language Steward
- Earn ¥3,000 / year platform credit (Steward-exclusive)
